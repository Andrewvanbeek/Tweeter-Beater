Videogame that takes real live tweets using the Tweeter api to allow users to search for any hashtag subject and select a random tweet from that subject to and place them into the a phaser a game, so users can interact with that actual tweet.  

Things to Note if you clone this you will need your own Twitter Api key.

For investors, this was a proof of concept I could easily take realtime twitter tweets and place them into a game, I am currently working on a more refined version.

Frameworks is rails and the phaser game engine as of now.  (if you are curious currently most of phaser logic resides in the view games/index)
